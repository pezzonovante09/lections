git init - инициализирует пустой локальный репозиторий, понять инициализирован ли локальный
репозиторий, можно при помощи команды ls -a (если создан, то покажет файл .git)

git add. (либо файл/директория вместо '.') - добавляет все файлы/директории в индекс гита 
(отслеживает все файлы/директории)

git commit -m '<сообщение для коммита>' - создает новую верисю, если были изменения в отслеживающихся 
файлах/директориях

git remote add <origin> SSH-link|HTTPS-link - устанавливает связь с удаленным репозиторием на github

git push origin <название ветки> - отправляет все изменения которые сохранил в коммите на 
удаленный репозиторий

git clone HTTPS-link|SSH-link - клонирует проект из удаленного репозитория на github на ваш компьютер
(локальный репозиторий)

git branch - показывает все ветки которые есть, и на какой ветке вы сейчас находитесь

git checkout <название ветки> - переходит на другую ветку
